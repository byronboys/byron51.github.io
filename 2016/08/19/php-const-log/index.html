PHP 日志常量

发表于 2016-08-19   |   分类于 PHP   |  

PHP中定义7个日志类型常量。

LOG_EMERG

系统不可用

LOG_ALERT

必须立刻采取行动 例如：在整个网站都垮掉了、数据库不可用了或者其他的情况下，应该发送一条警报短信把你叫醒。

LOG_CRIT

紧急情况 例如：程序组件不可用或者出现非预期的异常。

LOG_ERR

运行时出现的错误，不需要立刻采取行动，但必须记录下来以备检测。

LOG_WARNING

出现非错误性的异常。例如：使用了被弃用的API、错误地使用了API或者非预想的不必要错误。

LOG_NOTICE

一般性重要的事件。

LOG_INFO

重要事件 例如：用户登录和SQL记录。

LOG_DEBUG

调试信息
