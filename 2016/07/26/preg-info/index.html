正则表达式简介

发表于 2016-07-26   |   分类于 PHP ， 正则表达式   |  

边界符

  • / / 默认
  • # #
  • { }

原子

  • 可见原子 键盘输入我们肉眼可以看到的
  • 不可见原子 换行符\n 回车\r 制表符\t 空格 \f \v

元字符

  • | 标示匹配两个或者多个分支
  • [ 3 ] 匹配方括号中的任意字符
  • [^ ] 匹配除方括号中的任意字符

原子集合

  • . 匹配除换行符(\n)之外的任意字符
  • \d 匹配所有数字 [0-9]
  • \D 匹配非数字 [^0-9]
  • \s 匹配不可见原子 [\f\n\r\t\v]
  • \S 匹配可见原子 [^\f\n\r\t\v]
  • \w 匹配所有字母、数字和下划线 [0-9a-zA-Z_]
  • \W 匹配非字母、数字和下划线[^0-9a-zA-Z]

量词

  • {n} 匹配之前的字符出现n次
  • {n,} 匹配之前的字符至少出现n次
  • {n,m} 匹配之前出现的字符至少出现n次，最多出现m次
    • 出现0次，1次或者n次 {0,}
    • 至少出现1次 {1,}
  • ? 没有出现或者最多出现1次 {0,1}

边界控制

  • ^ 匹配开头位置
  • $ 匹配结尾位置

模式单元

  • ( ) 匹配其中的整体为一个原子 （D|d)uang

修正模式

当匹配的字符串产生歧义的时候，修正模式就发挥作用了，修正模式可以连用

  • U 懒惰模式
  • u 贪婪模式
  • i 忽略字母大小写
  • x 或略空白
  • s 忽略回车
